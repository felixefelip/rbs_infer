# Enumerize RBS Generator

Generates RBS type signatures for models that use the [enumerize](https://github.com/brainspec/enumerize) gem.

Instead of relying on the generic `Enumerize::Value` type from `gem_rbs_collection`, this generator creates **model-specific value classes** with the exact predicate methods defined by each enumerize attribute.

## Setup

Add `rbs_infer` to your `Gemfile`:

```ruby
gem "rbs_infer"
```

The rake task `rbs_infer:enumerize:all` is automatically available via Railtie — no manual file creation needed.

## Usage

```bash
rake rbs_infer:enumerize:all
```

Generated files are written to `sig/rbs_enumerize/`, following the same directory structure as `app/models/`.

## Example

Given a model:

```ruby
class Post < ApplicationRecord
  extend Enumerize

  enumerize :status, in: [:draft, :published, :archived], default: :draft, predicates: true, scope: :shallow
  enumerize :priority, in: [:low, :medium, :high]
end
```

The generator produces `sig/rbs_enumerize/app/models/post.rbs`:

```rbs
# Generated by rbs_infer (enumerize)

class ::Post::EnumerizeStatusValue < Enumerize::Value
  def draft?: () -> bool
  def published?: () -> bool
  def archived?: () -> bool
end
class ::Post::EnumerizePriorityValue < Enumerize::Value
  def low?: () -> bool
  def medium?: () -> bool
  def high?: () -> bool
end

class ::Post
  extend Enumerize::ActiveRecordSupport

  def status: () -> Post::EnumerizeStatusValue
  def status=: (String | Symbol | nil) -> (String | Symbol | nil)
  def status_text: () -> String?
  def status_value: () -> String?

  def draft?: () -> bool
  def published?: () -> bool
  def archived?: () -> bool

  def self.draft: () -> Post::ActiveRecord_Relation
  def self.not_draft: () -> Post::ActiveRecord_Relation
  def self.published: () -> Post::ActiveRecord_Relation
  def self.not_published: () -> Post::ActiveRecord_Relation
  def self.archived: () -> Post::ActiveRecord_Relation
  def self.not_archived: () -> Post::ActiveRecord_Relation

  def priority: () -> Post::EnumerizePriorityValue?
  def priority=: (String | Symbol | nil) -> (String | Symbol | nil)
  def priority_text: () -> String?
  def priority_value: () -> String?

end
```

## What It Generates

### Value type class

For each `enumerize` call, a specific class is generated inheriting from `Enumerize::Value`:

```rbs
class ::Post::EnumerizeStatusValue < Enumerize::Value
  def draft?: () -> bool
  def published?: () -> bool
  def archived?: () -> bool
end
```

The class name follows the pattern `Model::Enumerize<Attr>Value` (e.g., `Post::EnumerizeStatusValue`).

This enables type checkers to resolve method chains like `status.archived?` → `bool`.

### Instance methods

| Method | Return type | Description |
|--------|-------------|-------------|
| `status` | `Post::EnumerizeStatusValue` or `Post::EnumerizeStatusValue?` | Getter. Non-nilable when `default:` is present. |
| `status=` | `String \| Symbol \| nil` | Setter. |
| `status_text` | `String?` | Translated text for the current value. |
| `status_value` | `String?` | Raw string value. |

### Predicate methods (with `predicates: true`)

| Method | Return type |
|--------|-------------|
| `draft?` | `bool` |
| `published?` | `bool` |
| `archived?` | `bool` |

### Scope methods (with `scope:`)

#### `scope: :shallow`

| Method | Return type |
|--------|-------------|
| `self.draft` | `Post::ActiveRecord_Relation` |
| `self.not_draft` | `Post::ActiveRecord_Relation` |

#### `scope: true` (deep)

| Method | Return type |
|--------|-------------|
| `self.with_status` | `Post::ActiveRecord_Relation` |
| `self.without_status` | `Post::ActiveRecord_Relation` |

## Supported Options

| Option | Effect |
|--------|--------|
| `in:` | Defines the enum values and predicate methods on the value class. |
| `default:` | When present, the getter returns a **non-nilable** type. |
| `predicates: true` | Generates predicate instance methods on the model. |
| `predicates: { prefix: true }` | Prefixes predicates with attribute name (e.g., `status_draft?`). |
| `predicates: { only: [...] }` | Generates predicates only for the specified values. |
| `predicates: { except: [...] }` | Generates predicates for all values except the specified ones. |
| `scope: :shallow` | Generates named scopes per value (e.g., `self.draft`, `self.not_draft`). |
| `scope: true` | Generates `self.with_<attr>` and `self.without_<attr>` scopes. |
| `multiple: true` | Getter returns `Enumerize::Set` instead of the value class. |

## How It Works

The generator uses [Prism](https://github.com/ruby/prism) to parse model source files statically — no Rails boot required. It:

1. Scans all `app/models/**/*.rb` files
2. Finds `enumerize` method calls in class bodies
3. Extracts the attribute name, values, and options from the AST
4. Generates the corresponding RBS declarations

This means it works purely from source code, without loading your application.
